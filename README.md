# mdMind

## 🇺🇸 English Description
- *Two-Way Real-Time Sync* : A Markdown-based mind mapping plugin that delivers seamless, two-way real-time synchronization between Markdown text and the mind map view.
- *Hotkey Navigation* : Easily navigate through mind map nodes using the arrow keys (Up/Down/Left/Right).
- *Smart Callout/Comment Recognition* : Built-in support for notes and callouts, intelligently recognizing and rendering comment sections from your Markdown.
- *Impactful Visual Animations* : Features dynamic, high-impact node animations designed to enhance understanding and memory retention.
- *Full Node Editing* : Fully supports intuitive Add, Delete, and Edit operations directly within the map.

## 🇨🇳 中文介绍
- *双向实时同步* ：以 Markdown 为数据基础的思维导图插件，实现 Markdown 文本与思维导图视图之间的双向实时同步。
- *快捷键导航* ：支持使用上下左右方向键快速在导图节点间进行导航。
- *智能注释识别* ：支持注释功能，能够智能识别并渲染 Markdown 中的注释部分。
- *流畅视觉动画* ：导图节点拥有极具视觉冲击力的动画效果，帮助更直观地理解和记忆知识结构。
- *完整节点操作* ：完美支持节点的高效增加、删除、修改等编辑操作。

## ⚙️ 安装指南 / Installation

### 🇨🇳 社区市场安装（推荐）
1. 打开 Obsidian 的 **设置 (Settings)** -> **第三方插件 (Community plugins)**。
2. 关闭 **安全模式 (Safe mode)**。
3. 点击 **浏览 (Browse)**，搜索 `mdMind`。
4. 点击 **安装 (Install)**，安装完成后点击 **启用 (Enable)**。

### 🇺🇸 Via Obsidian Community Plugins (Recommended)
1. Open Obsidian **Settings** -> **Community plugins**.
2. Turn off **Safe mode**.
3. Click **Browse** and search for `mdMind`.
4. Click **Install**, then click **Enable** once the installation is complete.

### 🛠️ 手动安装 / Manual Installation
1. 前往 [Releases](https://github.com/Witt007/md-mind/releases) 页面下载最新的 `main.js`、`manifest.json` 和 `styles.css`。
2. 在你的 Obsidian 库目录中，进入 `.obsidian/plugins/`，新建一个名为 `md-mind` 的文件夹。
3. 将下载的三个文件放入该文件夹中。
4. 打开 Obsidian 的 **第三方插件** 设置面板，刷新并启用 `mdMind`。

## 📖 使用指引 / Usage Guide

### 🇨🇳 如何使用
1. **激活导图视图**：打开任意 Markdown 文件，点击编辑器右上角的 **思维导图图标**，或者使用命令面板（`Ctrl/Cmd + P`）搜索 `mdMind: Open Mind Map View`。
2. **双向同步编辑**：
   - 你可以直接在左侧的 Markdown 文本中修改大纲结构，右侧导图会**实时渲染**。
   - 也可以在右侧导图中双击节点进行编辑，或者使用快捷键增删节点，左侧的 Markdown 文本会**自动同步更新**。
3. **快捷键操作**：
   - `Arrow Keys (上下左右)`：在节点间快速导航。
   - `Tab`：为当前节点添加子节点 (Child Node)。
   - `Enter`：为当前节点添加同级节点 (Sibling Node)。
   - `Delete / Backspace`：删除当前节点。

### 🇺🇸 How to Use
1. **Activate View**: Open any Markdown file, click the **Mind Map Icon** in the top right corner of the editor, or open the Command Palette (`Ctrl/Cmd + P`) and search for `mdMind: Open Mind Map View`.
2. **Two-Way Sync**:
   - Edit the Markdown text hierarchy on the left, and the mind map on the right will **render in real-time**.
   - Alternatively, double-click a node or use hotkeys to edit/add/delete nodes on the right, and the Markdown text on the left will **sync automatically**.
3. **Hotkeys**:
   - `Arrow Keys`: Navigate swiftly between nodes.
   - `Tab`: Add a **Child Node** to the selected node.
   - `Enter`: Add a **Sibling Node**.
   - `Delete / Backspace`: Remove the selected node.
