#### 🇺🇸 English Description
* **Two-Way Real-Time Sync**: A Markdown-based mind mapping plugin that delivers seamless, two-way real-time synchronization between Markdown text and the mind map view.
* **Hotkey Navigation**: Easily navigate through mind map nodes using the arrow keys (Up/Down/Left/Right).
* **Smart Callout/Comment Recognition**: Built-in support for notes and callouts, intelligently recognizing and rendering comment sections from your Markdown.
* **Impactful Visual Animations**: Features dynamic, high-impact node animations designed to enhance understanding and memory retention.
* **Full Node Editing**: Fully supports intuitive Add, Delete, and Edit operations directly within the map.

### 🌟 核心特性 / Core Features

#### 🇨🇳 中文介绍
* **双向实时同步**：以 Markdown 为数据基础的思维导图插件，实现 Markdown 文本与思维导图视图之间的双向实时同步。
* **快捷键导航**：支持使用上下左右方向键快速在导图节点间进行导航。
* **智能注释识别**：支持注释功能，能够智能识别并渲染 Markdown 中的注释部分。
* **流畅视觉动画**：导图节点拥有极具视觉冲击力的动画效果，帮助更直观地理解和记忆知识结构。
* **完整节点操作**：完美支持节点的高效增加、删除、修改等编辑操作。
